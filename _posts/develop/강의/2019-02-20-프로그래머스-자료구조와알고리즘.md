## 2강. 선형 배열(Linear Array)
### 실습1
- 목표: 
- 예시: 
```python
def solution(L, x):
    for i, v in enumerate(L):
        if v > x:
            L.insert(i, x)
            break
    else:
        L.append(x)
    return L
```
### 실습2
```python
def solution(L, x):
    answer = []
    for i, v in enumerate(L):
        if v == x:
            answer.append(i)
    if len(answer) == 0:
        answer.append(-1)
    
    return answer
```
