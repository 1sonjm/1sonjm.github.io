# 자료구조와 알고리즘

> 1강은 소개로 생략

## 2강. 선형 배열(Linear Array)
python 의 list는 자료형의 제한이 없음

O(1)
-   원소 덧붙이기  `.append()`
-   원소 하나를 꺼내기  `.pop()`

O(n)
-   원소 삽입하기  `.insert()`
-   원소 삭제하기  `.del()`

기타
-   원소 탐색하기:  `.index()`
### 실습1
목표: 
```python
def solution(L, x):
    for i, v in enumerate(L):
        if v > x:
            L.insert(i, x)
            break
    else:
        L.append(x)
    return L
```
### 실습2
목표: 
```python
def solution(L, x):
    answer = []
    for i, v in enumerate(L):
        if v == x:
            answer.append(i)
    if len(answer) == 0:
        answer.append(-1)
    
    return answer
```
